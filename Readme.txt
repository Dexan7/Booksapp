Тестовый мини-проект:
Создать SPA в котором будем реализовано приложение для покупки книг.
Функционал:
Выбор книг из списка
Поиск по Названию книги и Автору
Кнопка покупки, вывода/скрытия цены
При нажатии покупки - вывести модальное окно
При подтверждении или отказа от покупки, вывести об этом информацию
Проверка возможностей анимации в vue

Инуструменты:
vue.js
bootstrap4

Примечание:
Все написано в трех файлах
Желательно использовать webpack, для разбития на компоненты
